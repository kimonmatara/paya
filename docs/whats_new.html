What’s New¶

Version 0.2¶

Additions¶

  • A wide range of maths operators, spread across paya.plugtypes and paya.datatypes, with support for bidirectional mixing of values and plugs
  • Added pick() and asOffset() to Attribute16D
  • Added pick() and asOffset() to Matrix
  • get() returns more useful types
  • Contextual name management with paya.lib.names.Name and pipeline suffixes
  • Dodge conversion nodes with paya.lib.mathops.NativeUnits and paya.lib.mathops.nativeUnits()

  • paya.runtime interface
