paya.nodetypes.dependNode.DependNode

class DependNode(*args, **kwargs)
makeName(*elems, name=None, inheritName=True, suffix=None)

Generates a context-appropriate Maya name. Results will vary depending on whether this method is called on a class or on a node instance.

Parameters

  • *elems – one or more name elements

  • name/n (None, str, int, or list) – elements contributed via name keyword arguments; these will always be prepended; defaults to None

  • suffix/suf (None, bool, str) – if string, append; if True, look up a type suffix and apply it; if False, omit; defaults to autoSuffix

  • inheritName/inn (bool) – inherit names from Name blocks; defaults to True

Returns

The node name.

Return type

str
