driveTwistChain(twistChain, upAxis, globalScale, downAxis=None, startUpMatrix=None, endUpMatrix=None, maintainOffset=False, decompose=False, skipEnd=False)

Drives a twist chain that was generated by duplicating this chain and calling subdivide() or insertJoints() on the duplicate. This method has no return value.

Parameters

  • twistChain (Chain) – the chain to drive

  • upAxis (str) – the dominant ‘curl’ axis on this chain, e.g. ‘x’

  • globalScale (int, float, Math1D) – a global scaling factor to normalize stretch

  • downAxis/da (str, None) – if you already know this chain’s ‘down’ axis, specify it here; defaults to None

  • startUpMatrix/sum (None, list, tuple, paya.runtime.plugs.Matrix, paya.runtime.data.Matrix) – an optional alternative matrix to govern the start up vector (useful for effects like ‘fixing’ upper shoulder twist); defaults to None

  • endUpMatrix/eum (None, list, tuple, paya.runtime.plugs.Matrix, paya.runtime.data.Matrix) – an optional alternative matrix to govern the end up vector; defaults to None

  • maintainOffset/mo (bool) – preserve offsets when applying matrix connections; defaults to False

  • decompose/dec (bool) – drive by connecting into SRT channels instead of offsetParentMatrix; defaults to False

  • skipEnd/ske (bool) – don’t drive the end joint; defaults to False
