insertJointsAtRatios(ratios, skipNumberingIfSingle=True, startNumber=1, name=None)

Inserts joints at the specified ratios, ‘subdividing’ the chain. This is an in-place operation, and this Chain instance’s membership will be modified.

Parameters

  • ratios (list) – the ratios at which to insert joints

  • skipNumberingIfSingle/sns (bool) – if only one ratio is specified, don’t add numbering; defaults to True

  • startNumber/sn (int) – for joint names, the starting number; defaults to 1

  • name/n (None, str, int, list or tuple) – one or more name elements; defaults to None

Raises
