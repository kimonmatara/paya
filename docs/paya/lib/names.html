paya.lib.names¶

Node-naming utilities, usually called indirectly by custom class methods.

legalise(name)¶

Modifies a string so that Maya will accept it as a node name.

Parameters:name (str) – the string to conform
Returns:The Maya name.
conformElems(*elems, padding=None)¶

Cleans up user name elements, typically specified through *args and / or the name/n keyword argument.

Parameters:
  • elems (int, str) – one or more name elements, packed or unpacked
  • padding/pad (None, int) – optional padding depth for any integers; defaults to None
Returns:

The conformed elements.
Return type:

list
class Name(*elems, inherit=True, suffix=None, padding=None)¶

Context manager. Accumulates name elements hierarchically.

Parameters:
  • elems (int, str) – one or more name elements, packed or unpacked
  • padding/pad – optional padding depth for any integers; defaults to None
  • inherit/i (bool) – inherit from containing blocks; defaults to True
  • suffix/suf (None, bool) – optional override for config.autoSuffix; defaults to None
__init__(*elems, inherit=True, suffix=None, padding=None)¶

Initialize self. See help(type(self)) for accurate signature.

make(*elems, name=None, inherit=True, suffix=None, node=None, nodeType=None, padding=None)¶

Constructs Maya node names.

Parameters:
  • elems (int, str) – one or more name elements, packed or unpacked
  • name/n (None, str) – pass-through for any elements provided via a name/n keyword argument; these will always be prepended; defaults to None
  • padding/pad (None, int) – optional padding depth for any integers; defaults to None
  • inherit/i (bool) – inherit prefixes from Name blocks; defaults to True
  • suffix/suf (bool, str) – optional override for config.autoSuffix; if True, apply suffixes; if False, omit them; if a string, use it; defaults to None
  • node (PyNode, str, None) – ignored if nodeType is provided; a node to inspect for an appropriate suffix lookup; defaults to None
  • nodeType/nt (bool, None) – ignored if suffix is a string; if provided, will be used as the suffix lookup; defaults to None
Returns:

The name.
Return type:

str
