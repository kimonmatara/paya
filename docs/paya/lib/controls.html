class ControlShapesManager(owner, attrName)

Implements the controlShapes / cs interface attribute on Transform.

‘Control shapes’ comprise any non-intermediate curve or locator shape under a transform.

  • To copy shapes and / or color, use copyTo().

  • To transform shapes, use rotate() and scale().

  • To assign shapes from the library, use setFromLib().

  • To save shapes into the library, use addToLib().

  • To interactively cycle through shapes, call cycle() repeatedly.

copyTo(*destControls, replace=True, worldSpace=False, mirrorAxis=None, color=None, shape=None)

Copies control shapes and / or color from this control to one or more destination controls. The ‘shape’ and ‘color’ flags can be defined by omission, Maya-style; for example, to copy only color, just pass col=True, and ‘shape’ will be set to False automatically unless explicitly set.

Parameters

  • shape/sh (bool) – copy shapes; if this is False, all other arguments except color are ignored

  • color/col (bool) – copy color

  • *destControls (list, str, Transform) – one or more controls to copy shapes to

  • replace/rep (bool) – remove existing shapes on the destination controls; defaults to True

  • worldSpace/ws (bool) – copy shapes in world space; defaults to False

  • mirrorAxis/ma (None, str) – a positive axis to flip when copying in local space, or along which to mirror when copying in world space, for example ‘x’; defaults to None

Returns

The new control shapes.

Return type

list of Shape

createControl(name=None, under=None, shape='cube', size=1.0, color=None, worldMatrix=None, keyable=None, channelBox=None, rotateOrder='xyz', asControl=True, offsetGroups='offset', pickWalkParent=None)

Creates rig controls. Available on paya.runtime.

Parameters

  • name/n (None, str, int, list) – one or more name elements

  • under/u (None, str, Transform) – an optional parent for the control; defaults to None

  • shape/sh (str) – the name of a library shape to apply to the control; defaults to ‘cube’

  • color/col (int, None) – an optional override color index for the control; defaults to None

  • size/sz (float) – a uniform scaling factor for the control shape; defaults to 1.0

  • worldMatrix/wm (list, Matrix, None) – the world matrix for the control; if this is omitted then, if under is provided, the matrix is copied from the destination parent; otherwise, it defaults to the identity matrix

  • keyable/k (list) – a list of channels to set to keyable on the control

  • channelBox/cb (list) – a list of channels to set to settable on the control

  • rotateOrder/ro (str, int) – the control’s rotate order; defaults to ‘xyz’

  • asControl/ac (bool) – if this is False, create the control as a group with no shapes and omit the controller tag; defaults to True

  • offsetGroups (list, str) – the suffixes of one or more offset groups to create; defaults to ‘offset’

  • pickWalkParent/pwp (str, DependNode) – an optional pick-walk parent for this control; ignored if asControl is False; defaults to None

Returns
