decomposeAndApply(transform, translate=None, rotate=None, scale=None, shear=None, compensatePivots=False, compensateJointOrient=True, compensateRotateAxis=True, compensateJointScale=True, fast=False)

Decomposes and applies this matrix to a transform.

Parameters

  • transform (str, Transform) – the transform node to drive

  • translate/t (bool) – apply translation

  • rotate/r (bool) – apply rotation

  • scale/s (bool) – apply scale

  • shear/sh (bool) – apply shear

  • fast (bool) – skip all compensations; defaults to False

  • compensateJointScale/cjs (bool) – account for segmentScaleCompensate on joints; defaults to True

  • compensateJointOrient/cjo (bool) – account for jointOrient on joints; defaults to True

  • compensateRotateAxis/cra (bool) – account for rotateAxis, set this to False to emulate Maya constraint behaviour; defaults to True

  • compensatePivots/cp (bool) – compensate for pivots (non-joint transforms only); this is expensive, so defaults to False

Returns

For convenience, the return of decompose() is passed along

Return type

dict
