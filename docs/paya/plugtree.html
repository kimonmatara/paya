paya.plugtree¶

Manages the inheritance tree used to construct the abstract paya attribute subtypes. The tree is defined inside paya/plugtree.json.

This module is not intended for direct use.

getPath(typeName)¶

Returns hierarchical type information.

Parameters:typeName (str) – the plug type to query, for example AttributeDoubleLinear.
Returns:A list of abstract type names, ordered similarly to nodeType() called with inheritance=True.
Return type:list
enumerator(enum)¶

Helper for reverse lookups on enumerators such as MFn.type.

getMPlugPathString(mplug)¶

Returns the full DAG path to a given MPlug.

Parameters:mplug (OpenMaya.MPlug) – the MPlug instance to query
Returns:The DAG path to the MPlug.
Return type:str
getMPlugTypeInfo(mplug)¶

Returns plug type information on the given MPlug in a dictionary comprising the following keys:

plugType The basic MObject type, e.g. kNumericAttribute.
dataType If available, the type of data served.
numericType The numeric subtype (e.g. kFloat) if the plugType was kNumericAttribute.
getTypeFromMPlug(mplug, inherited=False)¶

Given an MPlug, returns a best-fit name from the abstract class tree.

Parameters:
  • mplug (OpenMaya.MPlug) – the MPlug to query
  • inherited (bool) – return a hierarchy path, defaults to False
Returns:

A full inheritance stack if inherited=True, otherwise a single string.
Return type:

str or [str]
