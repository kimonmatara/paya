paya.util.functions¶

class short(**mapping)¶

Decorator with keyword arguments, used to mimic Maya’s ‘shorthand’ flags.

Example:

@short(numJoints='nj')
def makeJoints(numJoints=16):
    [...]

# This can then be called as:
makeJoints(nj=5)

resolve_flags(*flags)¶

Used to mimic Maya flag behaviour.

If only a subset of flags are defined, then:

  • If the subset are a mixture of True / False, the other flags are set to False.
  • If the subset are all True or all False, the remaining flags are set to the opposite.

If all flags are None, all flags are set to True. If all flags are NOT None (i.e. all defined), they are all conformed to booleans.

Parameters:flags – one or more user flag arguments
Returns:The resolved flags.
Return type:tuple
