paya.config

class Config(*args, **kwargs)

Dict-like object, inserted into sys.modules as a module. Initial values are read from paya/config.json on startup. Edits are only valid for the current session. The object can also be used as a context manager, with temporary overrides passed-in as keyword arguments:

import paya.config as config

print(config['suffixNodes'])
# True

with config(suffixNodes=False):
    print(config['suffixNodes'])
# False

print(config['suffixNodes'])
# False
