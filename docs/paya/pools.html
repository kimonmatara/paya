paya.pools¶

Defines the template class interfaces served by the nodes, plugs, comps and data attributes on paya.runtime.

This module is not intended for direct use.

ispmcls()¶
Parameters:cls – The class to query
Returns:True if the class is a ‘vanilla’ PyMEL class, otherwise False.
Return type:bool
iscustomcls()¶
Parameters:cls – The class to query
Returns:True if the class is a paya class, otherwise False.
Return type:bool

Abstract. Defines base functionality for all template class pools.

Given a PyNode or PyMEL data type instance, returns a custom class suitable for assignment.

Parameters:inst – the PyNode or PyMEL data type instance
Returns:The custom class.
purge()¶

Clears the cache so that subsequent class lookups will trigger reloads.

class NodeClassPool¶

Serves custom classes for nodes.

class PlugClassPool¶

Serves custom classes for plugs (attributes) off of an inheritance tree defined inside paya/plugtree.json.

getFromPyMELInstance(inst)¶

Overloads ClassPool.getFromPyMELInstance() to implement MPlug inspections.

Parameters:inst – the PyNode or PyMEL data type instance
Returns:The custom class.
class CompClassPool¶

Serves custom classes for components.

class DataClassPool¶

Serves custom classes for data objects, for example Matrix instances returned by getMatrix().

getByName(clsname)¶

Overloads ClassPool.getByName() to prevent customisation of sensitive types such as pymel.core.datatypes.MatrixN by raising UnsupportedLookupError.

Parameters:clsname (str) – the name of the class to retrieve
Returns:The custom class.
getPoolFromPmBase(pmbase)¶

Given a PyMEL base class, returns the most appropriate class pool to source a swap-in from.
