Package ConfigurationÔÉÅ

Paya runtime configuration is managed using the paya.config object. Although this is imported as a module, it behaves like a dictionary:

import paya.config
print(config)
# {'patchOnLoad': True, 'suffixNodes': True, 'padding': 0, 'inheritNames': True}

The keys and values in paya.config are loaded from paya/config.json on startup. Edits to paya.config are only valid for the current session. The object can also be used as a context manager, with overrides passed-in as keyword arguments and reverted on block exit:

r.nodes.Joint.createNode(name='eye_joint')
# eye_joint_JOIN

with config(suffixNodes=False):
    r.nodes.Joint.createNode(name='eye_joint')
    # eye_joint

To make permanent changes to the Paya configuration, edit paya/config.json directly and restart Maya. For more information on customising node naming behaviour, see Node Names.
