Package Configuration¶

Flags¶

The following configuration flags are defined inside paya.config:

Flag Default Value Description
patchOnLoad True Call start() the first time paya.runtime is imported.
autoSuffix True Apply node suffixes automatically.

More options may be added in future Paya releases.

Note

Any changes to paya.config will require a Maya relaunch.

Node Suffixes¶

Paya node suffixes are defined inside paya/lib/suffixes.json. To change them, you can:

  • Edit the file directly and then call paya.lib.suffixes.load()
  • Use paya.lib.suffixes.learnFromScene() interactively and then call paya.lib.suffixes.dump()

Warning

If you’ve already written automation code that depends on default Paya behaviours, changing the base Paya configuration may break it.
