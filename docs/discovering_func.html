Discovering Functionality¶

Call help() on any Paya object to print the same information as found in the Module Documentation, along with details on inherited PyMEL methods and classes:

cam = r.PyNode('persp')
help(cam)

Looking for a specific method? Use dir() on the object class instead. Here’s an algorithm to print any method or attribute that includes the word ‘parent’:

cls = type(cam)

for name in dir(cls):
    if 'parent' in name.lower():
        print name

Python’s inspect gives you even more options. For example, to find the module where a method is defined:

import inspect
print(inspect.getfile(r.PyNode('persp').createNode))
# C:\Users\RobertSapolsky\My Documents\Maya\2022\scripts\paya\nodetypes\dependNode.py
