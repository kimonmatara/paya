===========
paya.config
===========

.. py:module:: paya.config

Defines package-level configuration flags. Currently, these are:

.. list-table::
   :widths: 25 25 50
   :header-rows: 1

   * - Flag
     - Default Value
     - Description
   * - ``patchOnLoad``
     - ``True``
     - Call :py:meth:`paya.runtime.Runtime.start` on import.

More options may be added in future releases.
