#####################
Package Configuration
#####################

=====
Flags
=====

The following configuration flags are defined inside :py:mod:`paya.config`:

.. list-table::
   :widths: 25 25 50
   :header-rows: 1

   * - Flag
     - Default Value
     - Description
   * - ``patchOnLoad``
     - ``True``
     - Call :py:meth:`~paya.runtime.Runtime.start` the first time :mod:`paya.runtime` is imported.
   * - ``autoSuffix``
     - ``True``
     - Apply :ref:`node suffixes <Automatic Suffixes>` automatically.

More options may be added in future Paya releases.

.. note::

    Any changes to :mod:`paya.config` will require a Maya relaunch.

.. _redefining node suffixes:

=============
Node Suffixes
=============

Paya :ref:`node suffixes <Automatic Suffixes>` are defined inside ``paya/lib/suffixes.json``. To change them, you can:

*   Edit the file directly and then call :func:`paya.lib.suffixes.load`
*   Use :func:`paya.lib.suffixes.learnFromScene` interactively and then call :func:`paya.lib.suffixes.dump`

.. warning::

    If you've already written automation code that depends on default Paya behaviours, changing the base Paya
    configuration may break it.
