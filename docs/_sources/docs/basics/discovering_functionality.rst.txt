*************************
Discovering Functionality
*************************

For a semi-structured introduction to Paya, visit the *Rigging* sections of this documentation in-order. For a flat
reference of every Paya class, see :doc:`Types </types_index>`. Inside Maya, call :func:`help` on any class or method to
print information from :doc:`Types </types_index>`, or use :func:`dir` for name lookups:

.. code-block:: python

    import paya.runtime as r

    with r:
        # Get full class documentation
        loc = r.PyNode('locator1')
        cls = loc.__class__
        help(cls) # prints information on paya.runtime.nodes.Transform

        # Get information on a specific method
        help(loc.createOffsetGroups)

        # Look for a method based on a hunch
        for name in dir(loc):
            if 'offsetGroup' in name.lower():
                print(name)

        # Result: createOffsetGroups
