*************************
Discovering Functionality
*************************

Call :func:`help` on any Paya object to print the same information as found in the *Module Documentation*, along with
details on inherited PyMEL methods and classes:

.. code-block:: python

    cam = r.PyNode('persp')
    help(cam)

Looking for a specific method? Use :func:`dir` on the object **class** instead. Here's an algorithm to print any method
or attribute that includes the word 'parent':

.. code-block:: python

    cls = type(cam)

    for name in dir(cls):
        if 'parent' in name.lower():
            print name

Python's :mod:`inspect` gives you even more options. For example, to find the module where a method is defined:

.. code-block:: python

    import inspect
    print(inspect.getfile(r.PyNode('persp').createNode))
