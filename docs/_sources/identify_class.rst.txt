*****************************
Identifying Classes to Target
*****************************

The first step in object-orient development for Paya is to identify the right class to target.

The class of any Paya (or 'vanilla' PyMEL) object can be retrieved in two ways:

.. code-block:: python

    instance = p.PyNode('persp')

    # First way
    cls = type(instance)

    # Second way
    cls = instance.__class__

    print(cls)

    <class 'paya.nodetypes.transform.Transform'> # if patched
