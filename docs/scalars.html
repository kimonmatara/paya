General

Paya implements too much functionality on scalars to list here, from universal operators to methods for range management, comparisons, frame sampling and so on. For a full reference, see Math1D and Angles and Rotations.

Gating

The comparison modes of Maya’s condition node have been implemented as methods, for example gt(). These pair well with ifElse() to create logic gates in the dependency graph:

loc1.attr('ty').gt(2).ifElse(
    'polySphere1.output', 'polyCube1.output'
    ) >> mesh.attr('inMesh')
