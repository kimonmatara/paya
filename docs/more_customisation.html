More Customisation¶

Base Paya Configuration¶

The following configuration flags are defined inside paya.config:

Flag Default Value Description
patchOnLoad True Call paya.runtime.Runtime.start() on import.

More flags may be added in future releases.

Note

Changes to paya.config won’t be picked up until Maya is relaunched.

Runtime Interface Content¶

The runtime interface (paya.runtime) serves the entire content of paya.cmds which, in turn, imports the entire pymel.core namespace. Any functions or variables that you wish to make available directly on paya.runtime can therefore be added to paya.cmds.

Warning

When adding content to paya.cmds, take care not to shadow factory PyMEL commands from pymel.core. Doing so may break code that accesses such commands via paya.runtime.

Note

Changes to paya.cmds won’t be picked up until paya.runtime.Runtime.rehash() is called or Maya is relaunched.
