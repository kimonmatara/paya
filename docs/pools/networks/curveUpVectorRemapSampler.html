paya.runtime.networks.CurveUpVectorRemapSampler

class CurveUpVectorRemapSampler(*args, **kwargs)

Abstract base class for up vector samplers that interpolate using a remapValue node.

Inheritance

CurveUpVectorSampler Network Network DependNode DependNode PyNode ProxyUnicode

classmethod create(curve, keymap, interpolation='Linear')

Not for direct use; called by the subclasses.

Parameters

  • curve (str, paya.runtime.nodes.NurbsCurve, paya.runtime.nodes.Transform, paya.runtime.plugs.NurbsCurve) – the curve on which to create the sampler

  • keymap ([[float | str | Math1D], [tuple | list, paya.runtime.data.Vector, paya.runtime.plugs.Vector]]) – zipped parameter, vector pairs, indicating known up vectors along the curve; at least two such keys are needed

  • interpolation/i (int, str, Math1D) –

    a string label or integer value or plug specifying how to interpolate between the keyed pairs, namely:

    • 0 ('None') (you wouldn’t normally want this)

    • 1 ('Linear') (the default)

    • 2 ('Smooth')

    • 3 ('Spline')

Returns

The network system.
