blend(other, weight=0.5, swap=False, includeLength=False, byVectorAngle=False)
Parameters

  • other (list, tuple, paya.runtime.data.Vector, paya.runtime.plugs.Vector) – the vector that will be fully active when ‘weight’ is 1.0 (or 0.0, if ‘swap’ is True)

  • weight/w (float, Math1D) – the blending weight; defaults to 0.5

  • swap/sw (bool) – swap operands around; defaults to False

  • byVectorAngle/bva (bool) – blend by rotating one vector towards the other, rather than via linear value interpolation; defaults to False

  • includeLength/ilg (bool) – ignored if ‘byVectorAngle’ is False; blend vector lengths (magnitudes) as well; defaults to False
