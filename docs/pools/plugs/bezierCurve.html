paya.runtime.plugs.BezierCurve

class BezierCurve(*args, **kwargs)
getShapeMFn()

Returns an API function set for the shape type associated with this plug, initialised around the MObject of the data block. Useful for performing spot inspections (like numCVs() on a curve output) without creating a shape.

matrixAtAnchor(anchorIndex, tangentAxis, upAxis, upVector=None, aimCurve=None, closestPoint=True, globalScale=None, squashStretch=False)
Parameters

  • anchorIndex (int) – the anchor index

  • tangentAxis (str) – the axis to map to the anchor tangent, for example ‘-y’

  • upAxis (str) – the axis to map to the resolve up vector, for example ‘x’

  • upVector/upv (None, tuple, list, str, Vector, Vector) – an explicit up vector; defaults to None

  • aimCurve/aic (None, str, Transform, NurbsCurve, NurbsCurve) – an aim curve to derive up vectors from; defaults to None

  • closestPoint/cp (bool) – pull points from the aim curve based on proximity, not matched parameter; defaults to True

  • globalScale/gs (None, str, float, Math1D) – ignored if not a plug; a baseline scalar; defaults to None

  • squashStretch/ss (bool) – allow squash-and-stretch on the tangent vector; defaults to False

Returns

A matrix at the specified anchor.

Return type

Matrix
