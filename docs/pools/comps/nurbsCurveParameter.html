matrix(tangentAxis, upAxis, squashStretch=False, upVector=None, upObject=None, aimCurve=None, globalScale=None, matchedCurve=False, plug=False)
Parameters

  • tangentAxis (str) – the axis to align to the curve tangent

  • upAxis (str) – the axis to align to the resolved up vector

  • squashStretch/ss (bool) – incorporate tangent stretching (dynamic only); defaults to False

  • upVector/upv (None, list, tuple, str, Vector, Vector) – used as an up vector on its own, or extracted from upObject; defaults to None

  • aimCurve/aic (str, Transform, NurbsCurve, NurbsCurve) – an up curve; defaults to None

  • upObject/upo (None, str, Transform) – used as an aiming interest on its own, or as a source for upVector; defaults to None

  • globalScale/gs (None, float, Math1D) – used to drive scaling on dynamic matrices only; the scale will be normalised; defaults to None

  • matchedCurve/mc (bool) – set this to True when aimCurve has the same U domain as this curve, to avoid unnecessary closest-point calculations; defaults to False

  • plug/p (bool) – force a dynamic output; defaults to False

Returns

A matrix at the specified parameter, constructed using the most efficient DG configuration for the given options.
