Imports deformer weights for a specific shape from an XML file.

Parameters

  • filepath (str) – the path to the file (POSIX)

  • ignoreName/ig (bool) – see Maya help for deformerWeights(); defaults to None

  • method/m (str) – see Maya help for deformerWeights(); defaults to ‘index’

  • positionTolerance/pt (float) – see Maya help for deformerWeights(); defaults to None

  • remap/r (str) – see Maya help for deformerWeights(); note that this may be auto-populated to force a shape registration; defaults to None

  • shape – the shape that carries the weights; if omitted, a best- match shape is sought amongst the deformer members and the XML shapes; defaults to None

  • worldSpace/ws (bool) – see Maya help for deformerWeights(); defaults to None
