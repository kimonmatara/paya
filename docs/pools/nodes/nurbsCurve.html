class NurbsCurve(*args, **kwargs)
classmethod createFromMacro(macro, under=None, conformShapeNames=True, name=None, dispCV=False)

Recreates a curve from the type of dictionary returned by macro().

Parameters

  • macro – the macro dictionary to use

  • under/u (None, str, Transform) – an optional parent for the curve; defaults to None

  • conformShapeNames/csn (bool) – ignored if under is None; clean up destination shape names after reparenting to the specified transform; defaults to True

  • name/n (list, tuple, str) – one or more name elements; defaults to None

  • dispCV/dcv (bool) – display curve CVs; defaults to False

Returns

The curve.

Return type
